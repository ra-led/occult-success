# OccultSuccess iOS

SwiftUI MVP эзотерического приложения:

- лунный день на сегодня с фазой и короткой рекомендацией;
- детерминированный расчёт натальной карты с поиском города рождения, картой, координатами, часовым поясом, системами домов, круговой картой и LLM-расшифровкой через `openai/gpt-5.4`;
- сонник с генерацией через OpenRouter;
- раздел снятия порчи в формате безопасной бытовой практики;
- ритуалы на успех с прогрессом;
- “час успеха” с бесплатным периодом 3 недели, затем через StoreKit и локальные уведомления.

## Запуск

1. Откройте `Package.swift` в Xcode.
2. Выберите iPhone Simulator.
3. Для подписки создайте продукт `occultsuccess.success_hour.monthly` в StoreKit configuration или App Store Connect.
4. Для локального теста включите `Настройки -> Dev-доступ к часу успеха`.
5. Для сонника добавьте OpenRouter API key в настройках приложения. Base URL по умолчанию: `https://openrouter.ai/api/v1`, модель: `openai/gpt-5.4-mini`.

Для запуска из терминала после установки iOS Simulator runtime:

```bash
./scripts/run-simulator.sh
```

По умолчанию скрипт собирает приложение и запускает его на `iPhone 17`. Можно переопределить устройство:

```bash
SIMULATOR_NAME="iPhone 17 Pro" ./scripts/run-simulator.sh
```

## Важные MVP-ограничения

- Натальная карта считается детерминированно: город рождения ищется через системный Apple geocoder, дата и время интерпретируются в часовом поясе найденного города, Солнце/Луна/планеты считаются по геоцентрической эклиптической долготе от Julian Day и орбитальных элементов, асцендент и MC — через местное звёздное время, широту и долготу. Есть Whole Sign, Equal и итерационный Placidus, круговая карта, дома и мажорные аспекты. Swiss Ephemeris не встроен в репозиторий из-за AGPL/коммерческого лицензирования; для профессионального релиза его лучше подключать после покупки лицензии или через лицензированный backend.
- OpenRouter key в MVP вводится в приложении. Сонник использует модель из настроек, а LLM-расшифровка натальной карты использует `openai/gpt-5.4`. Для продакшена лучше вынести вызовы OpenRouter за backend-proxy, чтобы не раскрывать ключ в клиенте.
- “Час успеха” доступен 3 недели с первого запуска, затем требует подписку или dev-доступ.
- Уведомление “час успеха” показывается как лучший момент дня для действия.
- StoreKit-покупка подключена кодом, но требует настройки продукта вне репозитория.
