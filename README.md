# OccultSuccess iOS

SwiftUI MVP эзотерического приложения:

- лунный день на сегодня с фазой и короткой рекомендацией;
- базовый расчёт натальной карты;
- сонник с генерацией через OpenRouter;
- раздел снятия порчи в формате безопасной бытовой практики;
- ритуалы на успех с прогрессом;
- “час успеха” с бесплатным периодом 3 недели, затем через StoreKit и локальные уведомления.

## Запуск

1. Откройте `Package.swift` в Xcode.
2. Выберите iPhone Simulator.
3. Для подписки создайте продукт `occultsuccess.success_hour.monthly` в StoreKit configuration или App Store Connect.
4. Для локального теста включите `Настройки -> Dev-доступ к часу успеха`.
5. Для сонника добавьте OpenRouter API key в настройках приложения. Base URL по умолчанию: `https://openrouter.ai/api/v1`, модель: `openai/gpt-5.4-mini`.

Для запуска из терминала после установки iOS Simulator runtime:

```bash
./scripts/run-simulator.sh
```

По умолчанию скрипт собирает приложение и запускает его на `iPhone 17`. Можно переопределить устройство:

```bash
SIMULATOR_NAME="iPhone 17 Pro" ./scripts/run-simulator.sh
```

## Важные MVP-ограничения

- Натальная карта использует упрощённый расчёт без эфемерид и координат.
- OpenRouter key в MVP вводится в приложении. Для продакшена лучше вынести вызовы OpenRouter за backend-proxy, чтобы не раскрывать ключ в клиенте.
- “Час успеха” доступен 3 недели с первого запуска, затем требует подписку или dev-доступ.
- Уведомление планируется как случайное локальное уведомление в пределах ближайших 20 минут - 8 часов.
- StoreKit-покупка подключена кодом, но требует настройки продукта вне репозитория.
